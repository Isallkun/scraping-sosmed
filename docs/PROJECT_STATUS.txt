CONFIGURATION
================================================================================

Required in .env file:
  INSTAGRAM_USERNAME=your_username
  INSTAGRAM_PASSWORD=your_password
  INSTAGRAM_TARGET_URL=https://www.instagram.com/target_profile/

Optional settings:
  SCRAPER_HEADLESS=false              # true for background scraping
  SCRAPER_RATE_LIMIT=30               # requests per minute
  SCRAPER_LOG_LEVEL=INFO              # DEBUG for more details
  SENTIMENT_MODEL=vader               # or textblob

================================================================================
KNOWN LIMITATIONS
================================================================================

1. Placeholder Data
   - Simplified scraper uses placeholders for content/likes/comments
   - Workaround: Use post URLs to manually view full content

2. Instagram UI Changes
   - Instagram frequently updates their UI
   - Multiple fallback selectors provide resilience
   - May need updates for major UI changes

3. Rate Limiting
   - Instagram may block aggressive scraping
   - Best practice: Wait 5-10 minutes between sessions
   - Don't scrape more than 50 posts at once

4. Authentication
   - Instagram may require 2FA or CAPTCHA
   - Use an account that's logged in regularly
   - Run in non-headless mode first to verify login
